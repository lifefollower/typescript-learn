##  贪吃蛇游戏实例
### 项目依赖
**TypeScript**
-   typescript
-   ts-loader

**Webpack**
-   webpack
-   webpack-cli
-   webpack-dev-server
-   html-webpack-plugin
-   clean-webpack-plugin

**Babel**
-   core-js
-   babel-loader
-   @babel/core
-   @babel/preset-env

**Less & CSS**
-   style-loader
-   css-loader
-   less
-   less-loader
-   postcss
-   postcss-loader
-   postcss-preset-env

### 项目运行
**安装依赖**

`npm i`

**编译打包**

`npm run build`

**动态开发**

`npm run start`
